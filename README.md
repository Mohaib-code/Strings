## 💻 Implemented Problems

### 🔹 Palindrome Check

* Check if a string is same forward & backward
* Time Complexity: `O(n)`

### 🔹 Shortest Path

* Calculate shortest distance using directions
* Based on coordinate logic

### 🔹 String Compression

* Compress repeated characters (e.g., `aaabb → a3b2`)
* Efficient string traversal
